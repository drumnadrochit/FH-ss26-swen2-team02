## How To Run
### Starting the Backend
Docker Engine is needed for the backend.

1. Open the project folder conaining the **compose.yaml**
2. Start project with following command `docker compose up`

### Starting the Frontend
Nodejs and Angular CLI must be installed.

1. Navigate into following directory: ` ./frontend/tour-planner`
2. Install needed dependencies: `npm install`
3. Start frontend: `ng serve`

Now you should be able to use the tour-planner.
